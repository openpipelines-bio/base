Usage: NanoPlot [-h] [-v] [-t THREADS] [--maxlength MAXLENGTH]
                [--drop_outliers] [--downsample DOWNSAMPLE] [--loglength]
                [--readtype {1D,2D}] [--alength] [-c COLOR] [-o OUTDIR]
                [-p PREFIX]
                [-f {eps,jpeg,jpg,pdf,pgf,png,ps,raw,rgba,svg,svgz,tif,tiff}]
                [--plots [{kde,hex,dot} [{kde,hex,dot} ...]]]
                (--fastq [FASTQ [FASTQ ...]] | --fastq_rich [FASTQ_RICH [FASTQ_RICH ...]] | --summary [SUMMARY [SUMMARY ...]] | --bam [BAM [BAM ...]])

Perform diagnostic plotting, QC analysis Nanopore sequencing data and
alignments.

optional arguments:
  -h, --help            show this help message and exit
  -v, --version         Print version and exit.
  -t THREADS, --threads THREADS
                        Set the allowed number of threads to be used by the
                        script
  --maxlength MAXLENGTH
                        Drop reads longer than length specified.
  --drop_outliers       Drop outlier reads with extreme long length.
  --downsample DOWNSAMPLE
                        Reduce dataset to N reads by random sampling.
  --loglength           Logarithmic scaling of lengths in plots.
  --readtype {1D,2D}    Which read type to extract information about from
                        summary. Options are 1D or 2D
  --alength             Use aligned read lengths rather than sequenced length
                        (bam mode)
  -c COLOR, --color COLOR
                        Specify a color for the plots, must be a valid
                        matplotlib color
  -o OUTDIR, --outdir OUTDIR
                        Specify directory in which output has to be created.
  -p PREFIX, --prefix PREFIX
                        Specify an optional prefix to be used for the output
                        files.
  -f {eps,jpeg,jpg,pdf,pgf,png,ps,raw,rgba,svg,svgz,tif,tiff}, --format {eps,jpeg,jpg,pdf,pgf,png,ps,raw,rgba,svg,svgz,tif,tiff}
                        Specify the output format of the plots.
  --plots [{kde,hex,dot} [{kde,hex,dot} ...]]
                        Specify which bivariate plots have to be made.
  --fastq [FASTQ [FASTQ ...]]
                        Data is in default fastq format.
  --fastq_rich [FASTQ_RICH [FASTQ_RICH ...]]
                        Data is in fastq format generated by albacore or
                        MinKNOW with additional information concerning channel
                        and time.
  --summary [SUMMARY [SUMMARY ...]]
                        Data is a summary file generated by albacore.
  --bam [BAM [BAM ...]]
                        Data as a sorted bam file.
